Introduction

My personal project represents a piece of a human spine, in particular three lumbar vertebrae, the spinal cord inside of these and the spinal nerves. I chose to modeled three vertebrae because one in my opinion wasn’t enough, but the complete human spine was too much. So, I thought that three would have been the perfect number.

Project

The approach for modelling the project was to study the anatomy of the objects to be built. For this purpose some medical illustration taken from an anatomy book were used. After,these object were divided into different parts and modeled one part at a time. While modeling, the code has been separated in different functions for each part, named as the component it has meant to draw. For example, the function that builds the surface of the vertebral body is called 'VertebralBody' (in Italian 'CorpoVertebrale'). In this way it’s easy to understand the scope of the functions.
A ‘coordinate grid’ was necessary to help the developer to build the model and adjust it’s various parts. Moreover, the model is built using only Bezier surfaces (based on Bezier curves).

Model

First of all the components of a vertebra have been modeled. In order:

  • Vertical body
  • Spinous process
  • Articular processes
  • Transverse processes Than the Intervertebral disk was created and the various parts were adjusted to look for the best position and shape. The last things to be modelled were:
  • Spinal cord (with it’s various part)
    • Dura matter
    • White matter
    • Gray matter
  • Spinal nerves The last part of the project has been the merging of all the parts to compose the complete model. During the modeling process, in particular when building a new part of the model, I changed the alpha component of the model’s colors to 0,75. In this way I always can see through the model and adjust the various curves and surfaces.

Captions

To make the model similar to a medical illustration some captions were added to illustrate principal components of the model. To make this possible, a model for each of the 26 English characters was made (using the TRIANGLE_FAN function) and composed to build the captions. The result is a very beautiful effect.
